# Python3 Grammar

Source: https://github.com/antlr/grammars-v4/tree/master/python/python3

Grammar files required:
- `Python3Lexer.g4`
- `Python3Parser.g4`

To vendor these files, clone or download from the upstream repository and copy them here.

The grammar targets Python 3 syntax and is maintained by the antlr/grammars-v4 community.
Entry rule: `file_input`.
